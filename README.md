## Instatalks

### Description

Chat using Action Cable and Redis websockets

### Launch

1. Download or clone repo.

2. Use bundler

```console
$ bundle install
```

3. Run database migrations.

```console
$ rails db:create
$ rails db:migrate
```

4. Start server.

```console
$ rails s
```
